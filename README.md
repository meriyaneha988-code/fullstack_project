# Full Stack Ticket Booking Application

This is a Full Stack Ticket Booking Web Application with separate backend and frontend.

## Tech Stack

### Backend
- Node.js
- Express.js
- PostgreSQL
- REST API

### Frontend
- React.js
- Vite
- HTML, CSS, JavaScript

---

## Features

- User Registration & Login
- Event Listing
- Ticket Booking
- View Booked Tickets
- REST API Integration

---

## Project Structure

fullstack_project
│
├── ticket-booking-backend
└── ticket-booking-frontend

---

## Installation & Setup

### Backend

cd ticket-booking-backend  
npm install  
npm start  

Server will run on: http://localhost:3000

---

### Frontend

cd ticket-booking-frontend  
npm install  
npm run dev  

Frontend will run on: http://localhost:5173
